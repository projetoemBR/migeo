
1. install

sudo apt-get install git

--> normalmente ja instalado no ubuntu

2. configuration

git config --global user.name "usermame"
git config --global user.email "seuemail@mail.com"

--> usar suas infos para configurar seu git

3. initialize a repository

git init pasta

--> cria pasta com o nome que vc escolher e dentro dela uma pasta .git 
--> dentro da pasta .git fica as infos das modificacoes e commit

4. tracking changes

git status
--> verifica se tem ou nao mudanca no diretorio ou arquivos

git add


git commit
